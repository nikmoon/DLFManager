# DLFManager
 Managing downloaded files (via Torrent, DC, etc.)

 Программа предназначена для управления ссылками на загруженные файлы (каталоги).
 Например, можно скачивать и хранить в единственном каталоге различные файлы:
электронные книги, программы, фильмы и т.д. Для пользователя же удобнее, если
все эти файлы хранились в соответствующих каталогах, например Книги, Фильмы и т.п.
С помощью данной программы можно создавать в удобных каталогах ссылки на загруженные
файлы, удалять ссылки и т.п.
